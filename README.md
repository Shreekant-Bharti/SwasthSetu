# SwasthSetu - Bridging Healthcare for Rural India

SwasthSetu is a comprehensive healthcare platform designed to connect rural villages with quality medical services through smart appointments, online consultations, and integrated insurance solutions.

## Getting Started

This project requires Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone https://github.com/Shreekant-Bharti/SwasthSetu.git

# Step 2: Navigate to the project directory
cd SwasthSetu

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Features

- Smart appointment booking system
- Online consultations
- Integrated insurance solutions
- Multi-role dashboard (Patient, Doctor, Hospital, Pharmacy, Insurance, Admin)
- Rural healthcare connectivity
